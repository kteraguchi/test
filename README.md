AuthorizationKeyBehavior
===============

AuthorizationKeyBehavior Behavior

認証キー情報を登録、取得します。
モデル名、コンテンツID、認証キーをAuthorizationKeyモデルに登録します。
削除時も該当する認証キーデータを削除します。

#### サンプルコード
```
public $actsAs = array(
'AuthorizationKeys.AuthorizationKey'
);
```
